Code Example:

"String ends with?" KATA from Codewars

DESCRIPTION:

Complete the solution so that it returns true if the first argument(string) passed in ends with the 2nd argument (also a string).

Examples:

solution("abc", "bc"); // returns true solution("abc", "d"); // returns false

My Solution :

function solution(str, ending) { let end1Arg = str.length; let end2Arg = ending.length; let checkEnd = str.substring(end1Arg - end2Arg); return checkEnd === ending; }
